Edinburgh Napier University

The BEng(Hons) Cybersecurity and Forensics I am studying at Edinburgh Napier's Merchiston Campus; is full time and is the UK's first to be fully certified by the National Cyber Security Center (NCSC) focuses on vital areas of computer networks, security and forensics along with software development. The modules that comprise my program include:

Third Year

  • Networked Services
  • Network Security and Cryptography
  • Scripting for Cyberecurtiy and Networks
  • Web Technologies
  • Computing in Comtempory Society
  • Group Project

Fourth Year

  • OS Forensics
  • Security Systems for IoT
  • Security Testing and Advanced Network Forensics
  • Secure Software Development
  • Honours Project

The current presentation of the course provides an optional module for those who progrss from second year at Napier, but not for those who gain direct access into third year. The program is also acredited by the British Computer Society.
